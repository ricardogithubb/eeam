Capítulo 1: Hardware – A Base da Informática para o TST

O hardware é o conjunto de componentes físicos de um computador. Para o Técnico em Segurança do Trabalho, escolher o hardware correto impacta diretamente na produtividade para elaborar laudos, mapas de risco e análises estatísticas.

1.1 Componentes Internos Essenciais

ComponenteFunçãoRecomendação para TST
CPU (Processador)Executa cálculos e instruçõesIntel Core i5 (12ª geração+) ou AMD Ryzen 5 – mínimo 4 núcleos
Memória RAMArmazena dados temporários em uso8 GB mínimo (16 GB recomendado para AutoCAD e planilhas complexas)
Armazenamento (SSD/HD)Guarda arquivos permanentementeSSD de 256 GB ou mais (para velocidade na abertura de documentos e softwares SST)
Placa-mãeConecta todos os componentesCompatível com os demais componentes e com portas USB 3.0
Fonte de alimentaçãoFornece energia estávelMínimo 500W com certificação 80 Plus

1.2 Periféricos Indispensáveis

Impressora Multifuncional

Para imprimir laudos, CAT, mapas de risco e digitalizar documentos antigos.

Mouse e Teclado Ergonômicos

Previne LER/DORT – essencial para quem passa horas no computador.

Monitor 24" Full HD

Visualização confortável de plantas, gráficos e planilhas extensas.

Dica de ouro: Utilize um Nobreak para proteger seu equipamento contra quedas de energia e evitar perda de laudos não salvos.

Capítulo 2: Software – O que é e como classificar

Software é a parte lógica do computador – programas que realizam tarefas específicas. Divide-se em:

  • Software de Sistema: Sistemas Operacionais (Windows, Linux, macOS).
  • Software Aplicativo: Programas como Word, Excel, navegadores, softwares de gestão de SST (GSST, QRisk).
  • Software Utilitário: Antivírus, compactadores (WinRAR), backup.

Para o TST, dominar os aplicativos é tão importante quanto conhecer as NRs.

Capítulo 3: Sistemas Operacionais – Gerenciando o Computador

O Sistema Operacional (SO) é o software responsável por gerenciar o hardware e permitir a execução de outros programas.

Windows 10/11

Mais utilizado em empresas. Compatível com todos os softwares de SST (GSST, eSocial, QRisk).

Linux (Ubuntu)

Gratuito, seguro e leve. Alternativa com LibreOffice e softwares livres.

macOS

Utilizado em projetos de design de mapas e edição avançada.

Capítulo 4: Microsoft Word – Laudos, Ofícios e Documentos Técnicos

O Word é a principal ferramenta para elaborar laudos técnicos, ordens de serviço, comunicados internos, atas da CIPA e programas de treinamento.

4.1 Recursos Essenciais para o TST

Estilos e Formatação
Utilize estilos para Título 1, Título 2, Normal. Isso gera sumário automático (referência → Sumário).
Tabelas
Inserir tabelas para cronogramas de treinamento, controle de EPIs, registro de acidentes.
Inserir Imagens e Formas
Adicione plantas baixas, logotipos, setas indicativas em mapas de risco simplificados.
Salvar como PDF
Arquivo → Salvar como → PDF. Ideal para enviar laudos sem risco de alteração.

4.2 Template para Laudo de Insalubridade

Empresa: [Nome]

Dica: Use o recurso “Controlar Alterações” ao revisar documentos com a equipe. Nunca esqueça de salvar versões com data (ex: PPRA_JAN_2025_v2.docx).

Capítulo 5: Microsoft Excel – Estatísticas, Indicadores e Controle

O Excel é a ferramenta mais poderosa para o TST calcular indicadores, controlar EPIs, gerenciar prazos de exames e gerar gráficos profissionais.

5.1 Fórmulas Essenciais

  • SOMA – =SOMA(A1:A10) – Totaliza acidentes, horas trabalhadas.
  • MÉDIA – =MÉDIA(B1:B10) – Média de dias perdidos.
  • SE – =SE(C2>5;"ALERTA";"NORMAL") – Cria alertas automáticos.
  • PROCV – Procura valores em tabelas (ex: buscar nome do funcionário pelo código).

5.2 Cálculo da Taxa de Frequência (TF) – Exemplo Prático

Fórmula: TF = (Nº acidentes × 1.000.000) ÷ Horas trabalhadas

5.3 Gráficos no Excel

Selecione os dados → Inserir → Gráfico de barras, pizza ou linhas. Use para mostrar evolução de acidentes, distribuição de riscos ou comparação entre setores.

5.4 Tabela Dinâmica – Análise Rápida

Selecione os dados → Inserir → Tabela Dinâmica. Permite agrupar acidentes por setor, mês ou tipo em segundos. Indispensável para relatórios da CIPA.

Dica: Use o recurso “Formatação Condicional” para destacar células com valores acima do limite (ex: TF > 50 em vermelho).

Exercícios Práticos para Fixação

  1. Word: Crie um laudo de insalubridade simples com título, introdução, tabela de agentes e conclusão. Salve como PDF.
  2. Excel: Construa uma planilha com 5 meses de acidentes (dados fictícios) e calcule a Taxa de Frequência para cada mês. Insira um gráfico de barras.
  3. Mapa de Risco: No PowerPoint ou Canva, crie um mapa com 3 setores (Solda, Corte, Expedição) e insira círculos coloridos.
  4. Organização: Crie no seu computador a estrutura de pastas recomendada no Capítulo 3.
  5. Atalhos: Liste 5 atalhos de teclado que você mais usará no dia a dia.
Exemplo: Janeiro: 2 acidentes / 20.000h → TF = 100; Fevereiro: 3 / 18.000h → TF = 166,7. Gráfico: inserir barras.
